Happy Burger App

Programa de una franquicia de restaurantes para llevar la operación de pedidos de sus restaurantes.

Se utilizó la creación de un entorno env y la instalación de flask para mostrar datos en la web.
Se deberá crear el entorno de trabajo virtual con ayuda de los comandos python -m venv env, acceder a la carpeta Scripts y utilizar el comando .\activate para activar el entorno. Una vez allí se deberá regresar al entorno ../env/ e instalar flask con el comando pip install flask. Al término de este paso, se debe regresar a la carpeta donde se encuentra el proyecto y correrlo con el comando flask --app "nombreDelArchivoPrincipal" run.

Utiliza CRUD para soportar las bases de datos de clientes y productos y para añadir o eliminar pedidos. Imprime un ticket con la información del pedido generado en el momento y te permite continuar navegando a través del programa por las diferentes interfaces.
